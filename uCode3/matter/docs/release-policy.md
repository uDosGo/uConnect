# uHOME-matter Release Policy

`uHOME-matter` is a tagged optional-module repo in the v2 family.

## Tag When

- Matter bridge contracts change materially
- clone definitions or adapter mappings change materially
- checked-in examples or config surfaces change in a way downstream repos must
  consume

## Artifact Policy

- notes-only for now
- no placeholder binaries or exports

## Notes Source

- prefer `CHANGELOG.md`
- fall back to a generated summary when a tag section has not been cut yet
