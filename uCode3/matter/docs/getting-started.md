# uHOME-matter Getting Started

1. Read `docs/boundary.md`.
2. Review `docs/activation.md`.
3. Review `docs/architecture.md`.
4. Confirm this repo extends `uHOME-server` rather than replacing it.
5. Review `docs/server-runtime-handoff.md`.
6. Review `src/matter-clone-catalog.json` and `config/bridge-targets.example.json`.
7. Run `scripts/run-uhome-matter-checks.sh`.
8. Keep runnable integration examples in `examples/`.
9. Add tests for any contract or adapter change.
