# Reports Directory

## Purpose
This directory contains automated reports generated by the .dev flow system.

## Report Types

### Flow Execution Reports
- **Pattern**: `*-report-{{timestamp}}.md`
- **Generated by**: Various flows during execution
- **Content**: Execution details, outcomes, and metrics

### Synchronization Reports
- **Pattern**: `sync-report-{{timestamp}}.md`
- **Generated by**: doc-sync flow
- **Content**: Roadmap synchronization status and conflicts

### Coordinator Reports
- **Pattern**: `coordinator-{{timestamp}}.md`
- **Generated by**: Coordinator agent
- **Content**: Multi-agent workflow coordination metrics

### Quality Reports
- **Pattern**: `code-quality-{{timestamp}}.md`
- **Generated by**: CI/CD flow
- **Content**: Code analysis and quality metrics

## Report Retention Policy
- Daily reports: 30 days
- Weekly summaries: 90 days
- Monthly rollups: 1 year

## Usage
Reports are automatically generated by the flow system and can be:
- Reviewed manually in this directory
- Accessed via the reporting API
- Summarized in daily standup emails

## Automation
Reports are generated by:
- `ci-cd.flow.yaml` - Build and quality reports
- `doc-sync.flow.yaml` - Synchronization reports
- `coordinator.agent.yaml` - Coordination reports
