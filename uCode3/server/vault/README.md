# Vault

The `vault/` root is the Markdown-first sample data surface for the `uHOME`
pathway.

Today the active runtime still keeps local machine state under `memory/` and
JSON-backed config. This vault is the education-facing source of truth for the
pathway model:

- `homes/`
- `rooms/`
- `devices/`
- `schedules/`
- `media/`
- `notes/`

The examples here are intentionally small and Obsidian-compatible so the repo
can teach transparent household state before a full vault-backed runtime exists.
