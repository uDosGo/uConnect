# Services

The `services/` root is the education-facing map for runtime service responsibilities.

The active implementations still live under `src/uhome_server/`. This scaffold names the stable service lanes before any runtime code is moved.

Current service lanes:

- `playback/`
- `scheduling/`
- `launcher/`
- `lan-discovery/`
- `bridge/` as transitional runtime support for `uHOME-matter` extensions
