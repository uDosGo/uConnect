# uHOME-server Wiki

`/wiki` is the **beginner-friendly** lane for this repo, aligned with
[`uDOS-docs` `/wiki`](https://github.com/fredporter/uDOS-docs/tree/main/wiki).

Use it for:

- short orientation and checklists
- quick practical how-tos that are not full reference docs
- cross-links into `docs/` for depth

Use [`/docs`](../docs/README.md) for stable architecture, clients, operations,
and specifications.

Use [`/learning`](../learning/README.md) for pathway and course-style ordering
and for links into the **uDOS Library** (GitHub Pages).
