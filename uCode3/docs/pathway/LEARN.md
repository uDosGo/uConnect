# Learn uHOME Server

Use this path if you want to understand **uHomeNest** as the local-network
home infrastructure pathway (**uHOME** product) in the wider `uDOS` family.

## Start With

1. `README.md` at the repo root for the pathway contract
2. `courses/` for the student-facing lesson scaffold
3. `vault/` for the Markdown-first household examples

## Key Ideas

- Linux-first server runtime
- local-first household services
- LAN discovery and multi-node topology
- media and automation as teachable module families
- server contracts separated from native client-app repos
- online webhook and API automation is outside the uHOME stream; see `uDOS-wizard`
  and `uDOS-ubuntu` host docs rather than this lane
