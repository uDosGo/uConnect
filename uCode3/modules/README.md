# Modules

The `modules/` root is the capability map for optional or teachable `uHOME`
feature families.

These directories do not replace the live Python package yet. They provide the
stable language the repo should use when teaching and documenting the system.

Current module families:

- `media/`
- `dvr/`
- `home-ops/`
- `steam-surface/`
- `home-assistant-bridge/` as transitional local runtime support aligned to
  `uHOME-matter`
