# Courses

The `courses/` root is the student-facing learning path for the `uHOME`
pathway.

Current scaffold:

- `01-local-network-systems/`
- `02-media-infrastructure/`
- `03-home-automation/`
- `04-multi-device-systems/`
- `05-configurable-webhook-server/`

These courses are intentionally lightweight during Phase 1. They point learners
at the existing runtime surfaces and the new vault/module/service language
without pretending the full education system is already implemented.
