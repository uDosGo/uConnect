# v2.0.1 Client Alignment

`uHOME-client` is the lightweight public client-runtime layer that sits on top
of the `v2.0.1` platform spine.

## Contract Edges

- `uDOS-core` owns canonical command and workflow semantics
- `uDOS-shell` provides the shared shell adapter language for client-facing
  sessions
- `uDOS-wizard` supports assisted or remote routing when work leaves the local
  network
- `uHOME-server` owns the always-on local runtime consumed by these surfaces

## Starter Surfaces

- `src/runtime-profile-contract.json` for the baseline runtime-profile shape
- `src/runtime-profile-map.json` for the public client runtime-profile
  inventory

## Scope

This round does not make `uHOME-client` a full application runtime. It defines
the public contract and runtime-consumption layer that Android, iOS, kiosk,
controller, and remote surfaces will use as product work expands.

It is expected to operate both:

- as part of standalone `uHOME`
- as an integrated family component when `uDOS` is present
