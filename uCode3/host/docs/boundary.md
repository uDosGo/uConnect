# uHOME-client Boundary

## Owns

- lightweight client runtime modules
- local-network endpoint adapters and session shaping
- public example flows for runtime consumption

## Does Not Own

- always-on server services
- canonical runtime semantics
- Android or iOS UI and kiosk presentation
- private commercial app logic

## Deployment Rule

- support standalone `uHOME` operation
- support integrated `uDOS` operation when shared family services are present
