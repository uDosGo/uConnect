Troubleshooting

Common Issues

  • WASM failed to load: Check browser console for CORS errors. Try serving with python3 -m http.server 8000
  • Blank canvas: The WASM module might not have initialized. Check status above.
  • Slow performance: This is expected in debug mode. Build with --release for production.
