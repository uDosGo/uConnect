# How does crypto work? (Intake — optional path)

**Status:** Required **before** enabling **crypto mode** in uCoin-compatible apps.  
**Duration:** ~1 hour (self-paced).  
**Format:** Markdown modules + quizzes + practical demos (testnet).

## Outcomes

After completion, the learner:

- Can explain how **uCoin (barter)** differs from **cryptocurrency**.
- Understands **wallets, keys**, and why **custody matters**.
- Has completed a **risk acknowledgment**.
- Has used **Stellar testnet** demos where applicable.

1. Pass each module **quiz** (implementation may enforce in app).

3. Complete **testnet** wallet exercise (Module 5) where tooling exists.
