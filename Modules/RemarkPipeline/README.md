# @udos/remark-pipeline

Unified markdown processor for uDos CLI and editor surfaces.

## Commands (invoked by `udo md`)

- `format <file>`
- `lint <file>`
- `toc <file>`
- `render <file> --format html|teletext [--output file] [--terminal]`
- `frontmatter <file>`
- `check <file>`

## Custom plugins

- `remark-udos-obf`
- `remark-udos-usxd`
- `remark-udos-ucode`
- `remark-udos-teletext`
