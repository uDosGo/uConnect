# udos-template

**Role:** Canonical **vault skeleton** for uDos **v4.5.1 vault-native** work — see **[uDosDev `UDOS_VAULT_NATIVE_ARCHITECTURE_v4.5.1.md`](https://github.com/fredporter/uDosDev/blob/main/docs/specs/v4/UDOS_VAULT_NATIVE_ARCHITECTURE_v4.5.1.md)** (§2, §9–10).

**Path:** `vault/` in this directory mirrors **`~/Code/udos-template/vault/`** from the handoff.

Copy `vault/` into a new project or Obsidian workspace root; keep **`vault/.gitignore`** so **`vault/.udos/`** stays out of Git while remaining on disk.
