# The Quest for the Lost Link

**An interactive narrative that teaches `[[wiki-link]]` syntax**

*Part of the uDos Education Program — Public Lane*

---

## Overview

In this interactive story, you play as a young archivist who discovers that the
Great Knowledge Vault has been corrupted — all the `[[wiki-links]]` between notes
have been severed! You must journey through five chambers, each teaching a
different aspect of the uDos wiki-link system, to restore the connections.

## How to Play

1. Install the story pack: `udos student fetch --lane story --week quest-for-the-lost-link`
2. Run the story: `udos story play quest-for-the-lost-link`
3. Follow the prompts and type your answers to progress

## Learning Objectives

- Understand `[[wiki-link]]` syntax
- Create bidirectional links between notes
- Use link aliases: `[[target|display text]]`
- Navigate linked knowledge graphs
- Understand backlinks and forward links

## Story Structure

```
Chapter 1: The Broken Vault    → Introduction to links
Chapter 2: The Whispering Hall  → Creating your first links
Chapter 3: The Mirror Chamber   → Link aliases and display text
Chapter 4: The Web of Knowledge → Building a knowledge graph
Chapter 5: The Restoration      → Backlinks and navigation
```

## Files
