# Environment Variables

> **Note:** These are visible. Do not store production secrets here.
> For secrets, use system-level environment variables or a `.secrets` file (gitignored).

| Variable | Value | Purpose |
|----------|-------|---------|
| `GEMINI_API_KEY` | (empty) | Optional: for Nano Banana |
| `GITHUB_TOKEN` | (empty) | For auto-deploy |
| `UDOS_ENV` | development | development, production |
