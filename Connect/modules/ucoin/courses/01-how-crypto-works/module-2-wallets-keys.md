# Module 2 — Wallets, public keys, private keys

## Goals

- Explain **public** vs **private** material.
- State why **uDos does not custody** user keys in crypto mode.

## Reading

- **Public key / address:** safe to share for **receiving**.
- **Private key / seed:** **secret** — anyone with it can move funds.
- **Loss:** if you lose the private key and have no backup, assets may be **unrecoverable**.

## Demo (intent)

- Generate a **test** keypair in a **local** or **testnet** wallet — never reuse demo keys on mainnet.

## Quiz

1. **Which must you never paste into chat or email?**  
   *Answer:* Private key / seed phrase.

2. **Who holds your keys when crypto mode uses a wallet on your device?**  
   *Answer:* **You** — non-custodial.
