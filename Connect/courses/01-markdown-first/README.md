## The project — personal grimoire

Build a **vault** that tracks:

- **Contacts** — people in your world  
- **Projects** — quests  
- **Tasks** — spells to cast  
- **Notes** — lore  

**Final checkpoint:** Open the vault in Obsidian (or any markdown tool). Links resolve. Tasks parse. **Files remain source of truth** — no hosted DB required for the learning proof.

```
┌─────────────────────────────────┐
│   LEVEL 1 COMPLETE!             │
│                                 │
│   Markdown is data              │
│   Frontmatter is schema         │
│   Links are relationships       │
│                                 │
│   PRESS [SPACE] TO CONTINUE     │
└─────────────────────────────────┘
```

**YouTube hook:** *Build a database using only text files — and keep it yours.*
