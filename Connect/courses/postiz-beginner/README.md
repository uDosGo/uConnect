# Build Your Own Social Media Manager (Postiz beginner)

**Status:** locked outline — **Tool Family**, optional, educational (see [uDosDev `UDOS_TOOL_FAMILY_POSTIZ_NAME_GENERATOR_UCOIN_LOCKED_v1.md`](../../uDosDev/docs/specs/v4/UDOS_TOOL_FAMILY_POSTIZ_NAME_GENERATOR_UCOIN_LOCKED_v1.md)).  
**Upstream:** [gitroomhq/postiz-app](https://github.com/gitroomhq/postiz-app) — **AGPL-3.0**.  
**uDos posture:** uDos **does not** host Postiz; **does not** ship a uDos **SKIN**; **no** MCP automation mandate — human-in-the-loop.

## Outcomes

After this course, the learner can:

- Explain what Postiz is and why self-hosting matters for their threat model.
- Complete an **open-box config sheet** (domain, email, one social platform) before install.
- Clone and run Postiz on **their** infrastructure (Docker/VPS or equivalent — learner’s choice).
- Maintain updates, API keys, and backups **without** relying on uDos to operate the instance.
