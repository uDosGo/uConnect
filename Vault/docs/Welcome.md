# Welcome to Taskview

This is your **markdown document editor**. Start writing or open a file from the sidebar.

## Features

- **Typo-style toolbar** with flat SVG icons
- **Prose preview** powered by react-markdown
- **Vault-backed** storage at `~/Code/Vault/docs/`
- **Gift Wrapper** responsive font sizing
