# ThinUI MVP Spec

## Purpose

ThinUI is the preferred fullscreen, distraction-free primary surface for uDOS.

It is where the user should spend most focused time, especially on non-premium Mac hardware and Ubuntu hosts.

## Behaviour

- fullscreen by default
- one active surface
- no overlapping windows as the primary model
- no tab-heavy navigation

## Controls

Minimal persistent controls only:

- menu or dropdown menu bar
- close / escape control
- optional small status strip

## Layout

- primary markdown surface
- inline action areas
- expandable context areas
- optional side tray or slide-down utility area

## Interaction model

- binder-aware
- markdown-first
- expand/collapse instead of spawning windows
- minimal distractions
- compatible with launching normal OS tools when needed

## Design language

- Classic Modern tokens
- mono-first
- 1 or 2 accents only
- hard borders
- readable default text sizes
- checkerboard or subtle pattern backgrounds only where appropriate

## Non-goals

- dock-style desktop metaphor
- floating card stacks
- faux-mobile dashboard UI
- heavy animation
