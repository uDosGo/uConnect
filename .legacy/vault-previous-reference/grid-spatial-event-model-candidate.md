# Grid Spatial Event Model Candidate

Status: not yet roadmapped

## Candidate

Grid may later emit bounded canonical spatial events tied to place truth.

## Durable Idea

- place truth stays in Grid records
- events describe what happened in relation to that truth
- downstream systems may react differently without redefining place identity

## Event Families

- place events
- artifact events
- proximity and handshake events
- permission and security events

## Routing

If activated, route this into `uDOS-grid` first, not `uDOS-dev`.
