# Legacy Archive Retirement

Legacy archive mirrors are retired from public `@dev`.

## Rule

- old archive trees are not canonical documentation
- useful ideas must be distilled into current repo docs, wiki units, or
  forward-looking pathway notes
- public git should not carry mirrored historical source trees just for
  reference

## Outcome

- current family repos and their `docs/` trees are the public reference surface
- repo-local `@dev` holds only active planning and short candidate notes
- historical bulk stays local-only if it is still needed at all
