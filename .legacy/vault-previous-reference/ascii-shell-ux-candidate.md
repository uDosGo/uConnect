# ASCII Shell UX Candidate

Status: not yet roadmapped

## Candidate

Carry forward only these durable ideas:

- shell as a text workspace, not just a command line
- prompt-first and selection-first interaction can coexist
- stable block layout for menu, workspace, context, and prompt regions
- strong ASCII-native visual identity
