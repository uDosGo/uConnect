# UX surfaces in this repo

| Surface | Path / command | Role |
|--------|----------------|------|
| **Storybook workshop** | `npm run storybook` (port 6006) | Component stories, layout shells, kiosk placeholders |
| **Browser mockup** | `cd browser-mockup && npm run dev` | SvelteKit + Tailwind app prototype |
| **UX I/O lab** | Browser: `/lab/ux-io` in the mockup | Paste JSON bundles, pretty-print, download |
| **CLI import/export** | `npm run ux:export` / `npm run ux:import` | Token + story manifest for automation |

Swift and TUI notes live under `staff/swift/` and `staff/tui/` as handoff material, not runnable surfaces here.
