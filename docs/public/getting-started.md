# Getting started

1. Install Node.js **18+** and **npm 9+** (see root [README.md](../../README.md) Quickstart).
2. Run the **launcher** or **`sonic-express install`** — one workspace install at the repo root.
3. **`udo init`** then **`udo tour`** and **`udo doctor`**.
