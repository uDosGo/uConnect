# 👋 Welcome to uDos — Student Guide

Hey there! Welcome to **uDos**, a creative digital workshop where you can organise knowledge, build things, and learn at your own pace.

This guide is written in **plain language** — no jargon, no assumptions. If you're new to the command line or programming, you're in the right place.

## 🧠 What is uDos?

Think of uDos as a **digital workshop** with different tools for different jobs:

| Tool | What It Does |
|------|-------------|
| **Vault** | Your personal knowledge base (like a wiki on your computer) |
| **Binder** | Organises structured data (notes, feeds, configs) |
| **Surfaces** | Lets you create visual layouts (from simple text to 3D) |
| **Spools** | Automates repetitive tasks (like a smart to-do list) |
| **Courses** | Step-by-step lessons to learn new skills |

You don't need to understand all of this at once. Start with the **Vault** and **Courses** — the rest will make sense as you go.

## 🚀 Getting Started

### 1. Open Your Terminal

- **Mac**: Press `Cmd+Space`, type "Terminal", press Enter
- **Linux**: Press `Ctrl+Alt+T`

### 2. Clone the Repo

```bash
git clone https://github.com/uDosGo/Connect.git
cd Connect
```

### 3. Look Around

```bash
ls docs/        # Documentation
ls courses/     # Learning paths
```

### 4. Start a Course

```bash
cat courses/01-markdown-first/README.md
```
