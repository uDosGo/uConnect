# Markdown First Course

## Overview

This course introduces the basics of Markdown and how it's used in the uDos ecosystem.

## Lessons

- [Lesson 1: Introduction to Markdown](lessons/01-introduction.md)
- [Lesson 2: Markdown Syntax](lessons/02-syntax.md)
- [Lesson 3: Markdown in uDos](lessons/03-udos-integration.md)

## Objectives

By the end of this course, you will be able to:
- Write and format Markdown documents
- Use Markdown in uDos notes and vaults
- Apply Markdown templates and styles

## Prerequisites

- Basic computer literacy
- Familiarity with text editing
