# VA2 boundary (not VA1)

VA1 is **TypeScript only** (`udo` in this package). The following are **VA2** or later:

- **Go tools:** `ucode-cli`, `ucoin-ledger`, `meshcore-bridge`, game containers, ComfyUI bridge, …
- **TUI:** Bubble Tea or any full-screen terminal UI
- **Commands (examples):** `udo run`, `udo check`, `udo fmt` (uCode), `udo tui`, `udo ucoin`, `udo device scan`, `udo pkg install`, `udo world`, `udo portal`, `udo trade`, `udo mp`

Repo stubs under `tools/` remain placeholders until VA2 ships; **VA1 must not spawn them** from the `udo` CLI.
